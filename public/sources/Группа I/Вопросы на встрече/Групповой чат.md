# Предложение

- Для участников группы создаётся чат в телеграме
- Чат закрытый: новые участники не приглашаются, только если они не участвуют в группе
- Все анонсы, ссылки, материалы публикуются в чате после встречи
- Нет ограничений на флейм в чате: мемасики можно постить сколько влезет
- На чат распространяются правила безопасности группы
- В чате обсуждаются вопросы присутствия, предложения для будущих тем, уточнения по обсужденным темам
